Adult Income Analysis with Pandas

A data analysis project exploring the Adult Income dataset using Python and Pandas to identify demographic and socioeconomic factors influencing income levels.

Overview

This project analyzes census-based data to understand patterns behind income distribution, particularly the factors associated with earning more than $50K annually. Using data cleaning, exploratory data analysis (EDA), and visualization, the analysis highlights relationships between income and variables such as education, occupation, age, and working hours.

Objectives

Clean and preprocess the Adult Income dataset

Perform exploratory data analysis (EDA)

Examine relationships between income and demographic factors

Identify key predictors of higher income levels

Visualize patterns in income distribution

Tech Stack

Python

Pandas

NumPy

Matplotlib / Seaborn

Jupyter Notebook

Key Insights Explored
